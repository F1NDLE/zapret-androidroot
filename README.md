# Zapret for Android (Magisk/KernelSU Module)

Продвинутый модуль для обхода блокировок на Android, базирующийся на бинарнике nfqws (Zapret) и интегрированной системе RU AI UNLOCKER для доступа к нейросетям.

## Особенности
* RU AI UNLOCKER: Автоматическое обновление hosts для доступа к AI-сервисам (ChatGPT, Claude, Gemini и др.) напрямую из GitHub.
* Управление стратегиями: Поддержка нескольких пресетов в одном конфиге.
* CLI Утилита: Управление через терминал командой zapretfin.
* Автоматизация: Полная настройка iptables и маршрутизации при включении.
* Автозапуск: Возможность включать модуль автоматически при загрузке устройства.
* Чистый интерфейс: Отсутствие системного спама в логах KernelSU/Magisk.

## Структура модуля
- action.sh - основной скрипт управления и обновления hosts.
- service.sh - скрипт для обработки автозапуска при загрузке.
- autostart - файл-флаг (если существует, модуль включается сам при загрузке).
- strategy.txt - файл для выбора активной стратегии (просто цифра).
- common/config.txt - список стратегий.
- bin/nfqws - исполняемый файл Zapret.
- system/bin/zapretfin - команда для управления из терминала.

## Установка
1. Скачайте архив с модулем.
2. Установите через Magisk или KernelSU.
3. Перезагрузите устройство.

## Использование

### Через интерфейс
Нажмите кнопку Action в менеджере модулей. Скрипт проверит наличие обновлений для AI Unlocker и запустит Zapret с выбранной стратегией.

### Автозапуск
- Чтобы включить автозапуск: создайте пустой файл с названием `autostart` в папке модуля.
- Чтобы выключить: удалите файл `autostart`.

### Через терминал (Root)
```bash
su
zapretfin --zap     # Запустить Zapret
zapretfin --stop    # Остановить всё
zapretfin --status  # Проверить статус
