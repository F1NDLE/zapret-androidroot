# Zapret-android-root (Magisk/KernelSU Module)

Продвинутый модуль для обхода блокировок на Android, базирующийся на бинарнике nfqws (Zapret). Позволяет гибко переключать стратегии обхода прямо на лету.
Специальный модуль для разблокировки сервисов, которые крякнул РКН. Обходит благодаря nfqws.

## Особенности
* Управление стратегиями: Поддержка нескольких пресетов в одном конфиге.
* CLI Утилита: Управление через терминал командой zapretfin.
* Автоматизация: Полная настройка iptables и маршрутизации при включении.
* Чистый интерфейс: Никакого системного спама в логах KernelSU/Magisk.
* Гибкость: Поддержка списков (ipset, hosts, whitelist) и кастомных фейков (QUIC/TLS).

## Структура модуля
- common/config.txt - основной файл со стратегиями.
- strategy.txt - файл для выбора активной стратегии (просто цифра).
- bin/nfqws - исполняемый файл Zapret.
- scripts/ - логика запуска и сетевых правил.
- system/bin/zapretfin - системная команда для управления из терминала.

## Установка
1. Скачайте архив с модулем.
2. Установите через Magisk или KernelSU.
3. Перезагрузите устройство (для первой регистрации команды zapretfin).

## Использование

### Через интерфейс
Просто нажмите кнопку Action в менеджере модулей. Модуль запустит стратегию, номер которой указан в файле strategy.txt.

### Через терминал (Root)
Вы можете управлять модулем через любой эмулятор терминала (например, Termux):
```bash
su
zapretfin --zap     # Запустить Zapret
zapretfin --stop    # Остановить всё
zapretfin --status  # Проверить статус
